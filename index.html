About me ...

Sometimes it's a good thing when you have to reevaluate or change things in life. It can show you new exciting options you didn't thought about before. I wanted to use this opportunity to find something that could improve my work situation, but also challenge me in the years to come. So I knew it would not be the most easy way and that I would had to study hard for it.

What got my interest straight away, were some online courses with basic HTML and CSS. So I started doing them while I was still cheking all my possibilities. But after a few lessons I already knew this was it. And that I really wanted to do more with it.
Not realizing yet, but maybe feeling already, it would become a passion.

Skills.

HTML & CSS

Sass & Bootstrap

JavaScript

Of course I should normaly talk about things like semantics here. But during my courses I noticed that most students wanne learn programming languages asap. And with my passion for graphics both are also verry nice tools to work with.

Good HTML and CSS can be simple and even elegant, but really catch the users attention. And with each project I make, I try to achieve improve that. Make it compact, structured and comprehensible. And yes, with using the correct semantics. ðŸ˜‰

When we started using Sass in our first simple exercices, I was really convinced. Now I couldn't imagine not using it anymore. Not only for the extra's @mixin etc, but also just to organise and structure your stylesheets.

Probably I will never be a big fan of Bootstrap, but I learned to use it of course. My presonal impression is that those tools are making most websites look the same. You maybe don't really need a designer anymore, but it's also killing style and creativity. At least, that's my personal opiniion.

When starting with JavaScript it was the only short moment I doubted if web-developping was really my thing. I comprehended it (lke the best students in the group), but struggled with applying it. Luckely I just needed an 'eye-opener' like thet say. And DOM event hanndlings were my trigger I think.

From that moment I really started to enjoy JavaScript. Now everytime I learn something new or look something up, I wanne add things or use it for other applications, just to test myself extra. And because we learned ES5 in the courses, I'm also learning myself using ES6 better every project I make.
